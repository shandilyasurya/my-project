ğŸš€ How It Works

The user enters a query in the Nova interface.

The message is sent to the backend API endpoint /bot/v1/message.

The server validates input, logs it, and checks for a predefined reply.

If no match is found, it uses Gemini API to generate a contextual response.

The backend saves both user and bot messages in MongoDB.

The frontend dynamically displays both messages with smooth animations.
